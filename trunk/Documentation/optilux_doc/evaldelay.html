evaldelay

Evaluate the group-delay of the filter.

Syntax

Y=EVALDELAY(FTYPE,BW)

Description

Y=EVALDELAY(FTYPE,BW) evaluates the group delay of the filter FTYPE (see myfilter for available filters) having 3dB bandwidth BW.
Thanks to E. Forestieri for the filter expressions.
