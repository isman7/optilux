fprintmsg

Write a message into the file simul_out.

Syntax

FPRINTMSG(STR)

Description

FPRINTFMSG Write a message into the file simul_out. FPRINTMSG(STR) print the message STR into the file simul_out using the rules of FPRINTF (\n,\t,...).
